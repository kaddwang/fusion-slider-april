Node.js 安裝教學

🪟 Windows

  • 前往 nodejs.org 下載 LTS 版本
  • 執行 .msi 安裝檔，一路點擊 Next
  • 開啟 PowerShell 驗證安裝

🍎 macOS

  • 前往 nodejs.org 下載 LTS 版本
  • 執行 .pkg 安裝檔，完成安裝
  • 開啟 Terminal 驗證安裝
node -v # 應顯示 v18.x.x 或 v20.x.x
npm -v # 應顯示套件管理工具版本

Git 安裝教學

🪟 Windows

  • 前往 git-scm.com 下載安裝檔
  • 安裝時建議維持預設選項，一路點擊 Next

  • 開啟 Terminal 輸入 git --version
  • 若未安裝，系統會提示安裝 Command Line Tools
  • 點擊「安裝」即可完成，或使用 brew install git
git --version # 應顯示 git version 2.x.x

指示詞為何失準？

常見錯誤原因

❌ 缺乏背景

「幫我寫個網頁。」(什麼網頁？給誰看？)

❌ 假設 AI 有常識

「要做得好用一點。」(好用的定義因人而異)

❌ 步驟混亂

一次要求太多功能，導致 AI 顧此失彼。

精準指示詞模版

📸 建議拍照存擋

角色 (Role): 你是資深前端工程師...

輸出 (Output): 請提供完整的 index.html 與 script.js...

精準心法五大重點

  • 1. 限制技術棧：明確指定 (如：Tailwind + React)。
  • 2. 檔案結構優先：先定義資料夾結構，再寫程式碼。
  • 3. 小步迭代：一次只做一個功能 (One feature at a time)。
  • 4. Review Code：不要盲目複製，檢查 AI 的邏輯。
  • 5. Don't fight the AI：如果 AI 卡住，換個問法或開新對話。
